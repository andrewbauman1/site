<li class="note">
<div class="general-actions">
<h2 class="text">Homelab Setup</h2>
</div>
<div class="note-content">
<p>My personal infrastructure for testing, development, and automation running on local hardware. Docker-based services handle media management, photo backup, automation, and development environments. Network segmentation, secure remote access, and RAID storage mean I can experiment without turning "what if..." into a recovery project.</p>
<p>This is where I test configurations, build automation scripts, and learn new technologies before anything sees production. Currently working to expand automation workflows in my personal life.</p>
</div>
</li>

<li class="note">
<div class="general-actions">
<h2 class="text">Apple II Archive</h2>
</div>
<div class="note-content">
<p>Preserving programming manuals and documentation for the original <a href="https://en.wikipedia.org/wiki/Apple_II_(original)" target="_blank" rel="noopener noreferrer">Apple II</a>. I've restored an Apple II to working condition and am building a backlog of utilities and experiments in Applesoft BASIC and 6502 assembly. There's something satisfying about working within 48KB of RAM and figuring out how programmers solved problems with those constraints.</p>
<p>I'm planning to publish a repository of digitized manuals, code samples, and programming notes - the kind of documentation that's slowly disappearing from the internet too quickly. Equal parts preservation project and programming time machine. Partying like it's 1977.</p>
<p class="desc">Inspired by sites &amp; people like <a href="https://www.apple2history.org/" target="_blank" rel="noopener noreferrer">Apple2History</a>, <a href="https://archive.org/details/apple_ii_library_4am" target="_blank" rel="noopener noreferrer">Apple II Library 4am</a>, <a href="https://vintageapple.org/apple_ii/" target="_blank" rel="noopener noreferrer">VintageApple</a>, <a href="http://www.deater.net/weave/vmwprod/apple/" target="_blank" rel="noopener noreferrer">Vince Weaver</a></p>
</div>
</li>
